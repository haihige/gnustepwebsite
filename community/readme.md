# Community

The primary communication method is the mailing lists. Additional, develop-specific options can be found in [Developer](../docs/developer).

- discuss-gnustep: the most active list, for general discussion about using GNUstep and apps.

If you prefer forum-like communication, use our [GitHub Discussions](https://github.com/orgs/gnustep/discussions).

Chat with us over IRC in our `#gnustep` channel on the [Libera](https://libera.chat) network.

## Contributing

Find the source code on [GitHub](https://github.com/gnustep), where you can open a pull request to propose code contributions. Learn how to hack with Objective-C in [Developer](../docs/developer).

To report a bug or create a feature request, use our [issue tracker](https://github.com/gnustep/issues).
