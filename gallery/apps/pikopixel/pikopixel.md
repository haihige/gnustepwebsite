# PikoPixel

![](icon.jpg)

PikoPixel is an application for drawing and editing pixel art images.

PikoPixel is a free application for drawing & editing pixel-art images, originally for Mac OS X.
It supports multiple layers, customizable canvas background, hotkey-activated popup panels, and can export upscaled images

## Supported Platforms

| Mac | Windows | Linux | BSD |
|-----|---------|-------|-----|
| Yes | No | Yes | Yes |

Although Windows is not officially supported, PikoPixel may still be compiled for Windows.
